# Lab-Datos-UNSL

Este repositorio contiene el desarrollo de un proyecto académico basado en el dataset **SAT - Suicidios. Microdato (2017-2024)**. El objetivo del proyecto es analizar información detallada sobre suicidios a nivel nacional a partir de microdatos, con el fin de comprender patrones, contextos y características asociadas a los casos registrados.

## Fuente de datos

El dataset fue obtenido de la Dirección Nacional de Estadística Criminal del Ministerio de Seguridad de la Nación, a partir de bases de datos públicas de estadísticas criminales.

## Alcance del dataset

El conjunto de datos incluye información detallada sobre los hechos, las personas involucradas y el contexto de cada caso, permitiendo analizar variables como edad, modalidad, tipo de lugar y motivo del registro.
